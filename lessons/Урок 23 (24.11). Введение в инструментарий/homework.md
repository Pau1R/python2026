## 2) Установить на свой компьютер python и sublime text
1) [скачать python](https://www.python.org/ftp/python/3.13.9/python-3.13.9-amd64.exe)
2) запустить установщик из-под админа
3) Обязательно отметить вторую галочку "Add Python 3.8 to PATH":
<img width="409" height="294" alt="Screenshot 2025-11-24 at 23 47 39" src="https://github.com/user-attachments/assets/85b94db6-443b-4cde-882c-b689396e7213" />

4) Запустить установку кнопкой "Install Now"
5) Пройти все остальные этапы установки оставляя значения по умолчанию
6) [скачать sublime text](https://www.sublimetext.com/download_thanks?target=win-x64)
7) Установить sublime text со значениями по умолчанию.
8) Перезагрузить компьютер

### Проверка работы sublime text:

1) Открыть sublime text
2) Написать код:
```
print("hello, world!")
```
3) Запустить код нажав сочетание "ctrl+b"
4) Проверить результат работы в нижнем окне. Если там есть текст "hello, world!" - все установлено корректно

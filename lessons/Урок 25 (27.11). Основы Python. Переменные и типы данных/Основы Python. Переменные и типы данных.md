## Теоретическая часть

### 1. Зачем нужны типы данных
В Python каждая величина относится к определенному типу. тип определяет доступные операции и способ хранения значения.
- **int**. целые числа без дробной части.
- **float**. числа с плавающей точкой, используемые для дробных значений.
- **bool**. логический тип, принимающий значения `True` или `False`.
- **str**. строки, последовательности символов, заключенные в кавычки.
- **NoneType**. особый тип, представляющий отсутствие значения (`None`).

Типы помогают интерпретатору правильно выполнять операции. например, сложение чисел, конкатенация строк, сравнение значений.

### 2. Переменные и присваивание
Переменная представляет имя, которое связывается с объектом в памяти. В Python переменные объявляются в момент присваивания.
```python
age = 15      # целое число
height = 1.7  # число с плавающей точкой
name = "Alex" # строка
```
Имена переменных должны начинаться с буквы или символа подчёркивания и могут содержать буквы, цифры и `_`. Следуйте стилю `lower_case_with_underscores`.

#### Динамическая типизация
Python не требует указывать тип переменной заранее. интерпретатор определяет его автоматически по присвоенному значению. Переменная может ссылаться на объект любого типа и при необходимости изменять его:
```python
value = 10
value = "Python"  # теперь value становится строкой
```
Такое поведение удобно, но важно отслеживать, какие операции выполняются над переменной, чтобы не получить ошибку типа.

### 3. Преобразование типов
Для изменения типа значения используются функции-преобразователи:
- `int("42")` → 42
- `float("3.14")` → 3.14
- `str(5)` → "5"
- `bool(0)` → False, `bool(1)` → True

Будьте внимательны при преобразовании строк в числа: `int("3.14")` вызовет ошибку. Поэтому дробные значения сначала приводят к `float`, а уже затем при необходимости к `int` с потерей дробной части.

#### Типичные ошибки
- Попытка сложить число и строку: `"5" + 3` → `TypeError`.
- Ввод значения через `input()` без преобразования: `input()` всегда возвращает строку.

### 4. Ввод данных и базовые программы
Функция `input()` позволяет читать данные от пользователя. По умолчанию возвращает строку, поэтому нужно преобразовать значение к нужному типу:
```python
number = int(input("Введите целое число: "))
print(number + 5)
```

Для дробных значений: `float(input("Введите число: "))`.

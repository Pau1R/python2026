# Примеры кода к урокам 1-7

## Урок 1. Введение в Python и условные конструкции

### Проверка возраста
```python
user_age = int(input("Введите возраст: "))
message = "Доступ разрешён" if user_age >= 18 else "Требуется сопровождение"
if 60 <= user_age <= 120:
    message += ", пожалуйста, сделайте перерыв от экрана"
print(message)
```
- Программа запрашивает возраст пользователя.
- Условный оператор с тернарным выражением выбирает текст ответа.
- Дополнительная проверка добавляет предупреждение для пожилых пользователей.

### Присвоение оценок на основе баллов
```python
score = 78
grade = (
    "Отлично" if score >= 90
    else "Хорошо" if score >= 75
    else "Удовлетворительно" if score >= 60
    else "Нужно подтянуть материал"
)
print(grade)
```
- Вложенные тернарные выражения выбирают итоговую оценку в зависимости от баллов.
- Переменная `grade` хранит итоговый текст, который выводится на экран.

## Урок 2. Циклы: for и while. Исключения и обработка ошибок

### Деление с обработкой исключений в цикле for
```python
numbers = [10, 5, 0, 8]
for n in numbers:
    try:
        print(f"{n} → 100 / {n} = {100 / n:.1f}")
    except ZeroDivisionError:
        print(f"{n} → деление на ноль запрещено")
```
- Цикл перебирает список чисел и пытается вычислить частное.
- Конструкция `try/except` ловит деление на ноль и выводит понятное сообщение.

### Сумма чисел из ввода с защитой от ошибок
```python
total = 0
while True:
    raw = input("Введите число или stop: ")
    if raw.lower() == "stop":
        break
    try:
        total += float(raw)
    except ValueError:
        print("Это не число, попробуйте ещё раз")
print(f"Сумма: {total}")
```
- Бесконечный цикл собирает числа до ввода команды остановки.
- Ввод проверяется и преобразуется в число.
- Ошибки преобразования перехватываются, что не прерывает работу программы.

## Урок 3. Списки и их методы

### Расширение списка и удаление дублей
```python
courses = ["python", "git", "sql"]
courses.extend(["linux", "docker"])
unique_sorted = sorted(set(courses), key=courses.index)
print(unique_sorted)
```
- Метод `extend` добавляет элементы в конец списка.
- Преобразование в множество временно удаляет повторы.
- Сортировка с ключом `courses.index` возвращает исходный порядок без дублей.

### Фильтрация и медиана оценок
```python
grades = [5, 4, 3, 5, 2, 4]
filtered = [g for g in grades if g >= 4]
median = sorted(filtered)[len(filtered) // 2]
print(f"Фильтрованные оценки: {filtered}, медиана: {median}")
```
- Генератор списка оставляет только высокие оценки.
- Отсортированный список позволяет быстро взять медиану.

## Урок 4. Строки и их методы

### Нормализация текста
```python
text = "  привет, Мир!  "
clean = text.strip().lower().replace("мир", "python")
print(clean.title())
```
- `strip` удаляет пробелы, `lower` приводит к нижнему регистру, `replace` заменяет слова.
- Метод `title` форматирует строку с заглавной буквой каждого слова.

### Разделение почты на имя пользователя и домен
```python
email = "student@example.com"
user, domain = email.split("@")
print(f"Имя: {user}, домен: {domain.upper()}")
```
- `split` разбивает строку по символу `@` на две части.
- Домен приводится к верхнему регистру для выделения.

## Урок 5. Множества

### Операции над множествами студентов
```python
group_a = {"anna", "ivan", "maria"}
group_b = {"maria", "sergey", "ivan"}
print("Общий список:", group_a & group_b)
print("Только в группе A:", group_a - group_b)
```
- Пересечение множеств показывает общих участников.
- Разность множеств оставляет только уникальных для первой группы студентов.

### Разделение букв по признаку гласная или согласная
```python
letters = set("информатика")
vowels = {"а", "е", "ё", "и", "о", "у", "ы", "э", "ю", "я"}
print("Гласные:", letters & vowels)
print("Согласные:", letters - vowels)
```
- Множества символов позволяют быстро искать принадлежность.
- Операции пересечения и разности разделяют гласные и согласные.

## Урок 6. Кортежи и словари

### Словарь точек с неизменяемыми координатами
```python
point = (12.5, -3.2)
points = {"A": point, "B": (0, 0)}
points = points | {"C": (point[0] * 2, point[1] * 2)}
print(points)
```
- Кортеж хранит координаты, что гарантирует неизменяемость значения.
- Оператор `|` создаёт новый словарь с добавленной точкой `C`.

### Таблица результатов и поиск лидера
```python
scores = [("ivan", 87), ("anna", 92), ("oleg", 78)]
leaderboard = {name: score for name, score in scores if score >= 80}
top_name = max(leaderboard, key=leaderboard.get)
print(f"Лучший студент: {top_name}, балл: {leaderboard[top_name]}")
```
- Словарь создаётся из списка кортежей с фильтрацией по минимальному баллу.
- Функция `max` с ключом ищет имя студента с наивысшим результатом.

## Урок 7. Функции и модули

### Поиск расстояния между точками
```python
from math import hypot

def distance(p1, p2):
    return hypot(p2[0] - p1[0], p2[1] - p1[1])

print(distance((0, 0), (3, 4)))
```
- Функция `distance` использует `hypot` из модуля `math`, чтобы вычислить расстояние по двум координатам.
- Вызов функции показывает работу на простом примере.

### Статистика по набору значений
```python
import statistics as stats

def describe(*values):
    clean = [float(v) for v in values if v is not None]
    return {
        "count": len(clean),
        "mean": stats.mean(clean),
        "stdev": stats.pstdev(clean),
    }

print(describe(10, 12, None, 9, 15))
```
- Функция принимает произвольное количество аргументов и игнорирует `None`.
- Модуль `statistics` даёт среднее значение и стандартное отклонение.
